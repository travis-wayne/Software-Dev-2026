# Branch Demo Site

This is a simple multi-page HTML site used to practice Git branching, merging, and pull requests in Lesson 8.

## Project Structure

- `index.html` — The main page. Used for the merge conflict exercise.
- `about.html` — Used for adding new sections via branches.
- `contact.html` — Used for the feature branch exercise.
- `css/style.css` — Shared styles.
