Exercise 6: Buttons & Hover Effects

Style these buttons using Tailwind's hover and transition utilities.

6a. Primary Button

Add: bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-6 rounded transition duration-300

6b. Outline Button

Add: bg-transparent hover:bg-green-500 text-green-700 hover:text-white font-semibold py-2 px-6 border border-green-500 hover:border-transparent rounded transition duration-300

6c. Pill Button with Scale Effect

Add: bg-purple-600 hover:bg-purple-800 text-white font-bold py-2 px-6 rounded-full transform hover:scale-105 transition duration-300

7. Challenge — Build a Portfolio Section

Combine everything you've learned to build a mini portfolio section using only Tailwind classes.

📋 Requirements:

  • A heading for the section (e.g., "My Projects")
  • A responsive grid of at least 3 project cards
  • Each card should have a coloured header, title, description, and a link
  • Cards should be 1 column on mobile, 2 on tablet, 3 on desktop
  • Add hover effects to cards (shadow, scale, or colour changes)
  • Style a "View All" button below the grid
