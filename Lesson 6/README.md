# Lesson 6 – CSS Frameworks (Tailwind CSS) & Portfolio Styling

## 📚 Learning Objectives

By the end of this session, you will be able to:

1. Understand the concept and benefits of utility-first CSS frameworks like Tailwind CSS.
2. Set up Tailwind CSS in a basic HTML project.
3. Apply Tailwind CSS utility classes to style HTML elements.
4. Style your personal portfolio project using Tailwind CSS.

## 🔑 Key Concepts

- **CSS Frameworks** — Pre-built CSS systems that speed up development
- **Utility-First CSS** — Styling by composing small, single-purpose classes
- **Tailwind CSS** — A popular utility-first CSS framework
- **Responsive Prefixes** — `sm:`, `md:`, `lg:`, `xl:` for breakpoint-based styling
- **Hover & State Variants** — `hover:`, `focus:`, `active:` for interactive styles
- **Customisation** — Extending Tailwind's default design tokens
