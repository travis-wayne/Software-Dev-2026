0. Traditional CSS vs. Tailwind — Side by Side

The same result, two different approaches. Look at how much less code switching is needed with Tailwind.

❌ Traditional CSS (2 files)

HTML file:

<nav class="main-nav">
  <span class="nav-logo">MyBrand</span>
  <div class="nav-links">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
  </div>
</nav>

CSS file (separate!): 

.main-nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
  background-color: #1e293b;
  color: white;
}
.nav-logo {
  font-size: 1.25rem;
  font-weight: bold;
}
.nav-links { display: flex; gap: 1.5rem; }
.nav-links a:hover { color: #93c5fd; }

1. Spacing — Padding & Margin

Each number in Tailwind's spacing scale equals 0.25rem (4px). So p-4 = 1rem = 16px.

p-2 — 0.5rem (8px) padding
p-4 — 1rem (16px) padding
p-8 — 2rem (32px) padding

Directional Padding

px-8 py-2 — More horizontal padding
px-2 py-8 — More vertical padding

Margin — Auto Centering

w-1/2 mx-auto — 50% width, centred

2. Typography

Tailwind gives you utility classes for font size, weight, colour, and alignment.

text-xs — Extra small (0.75rem)

text-sm — Small (0.875rem)

text-base — Base / normal (1rem)

text-lg — Large (1.125rem)

text-xl — Extra large (1.25rem)

text-2xl — 2× extra large (1.5rem)

text-3xl font-bold — 3× extra large, bold

Font Weight

font-light — Light weight (300)

font-normal — Normal weight (400)

font-medium — Medium weight (500)

font-semibold — Semi-bold (600)

font-bold — Bold (700)

font-extrabold — Extra bold (800)
