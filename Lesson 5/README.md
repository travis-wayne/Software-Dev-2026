3. Create flexible and adaptable layouts that work across different screen sizes.

## 🔑 Key Concepts

- **CSS Layouts** — Block, Inline, Flexbox, and Grid display modes
- **Flexbox** — One-dimensional layout (rows OR columns) with `display: flex`
- **CSS Grid** — Two-dimensional layout (rows AND columns) with `display: grid`
- **Responsive Web Design (RWD)** — Making sites work on any screen size
- **Media Queries** — Applying styles conditionally based on screen width
- **Viewport Meta Tag** — Ensuring proper scaling on mobile devices
