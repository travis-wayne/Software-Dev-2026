# Lesson 2 – Advanced HTML5, Semantic Web, Forms & Validation

## 📚 Learning Objectives

By the end of this session, you will be able to:

1. Utilize advanced HTML5 elements for better page structure and functionality.
2. Understand and apply semantic HTML to improve accessibility and SEO.
3. Create various types of HTML forms with different input fields.
4. Implement basic HTML5 form validation.

## 🔑 Key Concepts

- **Semantic HTML5 Elements** — `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`, `<figure>`, `<figcaption>`
- **HTML Forms** — `<form>`, `<input>`, `<textarea>`, `<select>`, `<option>`, `<button>`, `<label>`
- **Input Types** — `text`, `email`, `password`, `number`, `date`, `checkbox`, `radio`
- **Form Attributes** — `action`, `method`, `name`, `value`, `placeholder`, `required`
- **HTML5 Validation** — Built-in browser validation with `required`, `type`, `min`, `max`, `pattern`
