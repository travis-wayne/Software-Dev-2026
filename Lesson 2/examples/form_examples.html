📋 HTML Form Input Types – Reference

This page demonstrates the most common HTML form elements and input types. Try filling them in and clicking "Submit" to see browser validation in action.

1. Text Input

Use <input type="text"> for general text.

2. Email Input

Use <input type="email"> — the browser validates the format.

3. Password Input

Use <input type="password"> — text is hidden as you type.

4. Number Input

Use <input type="number"> with min and max.

5. Date Input

Use <input type="date"> to show a date picker.

6. Radio Buttons

Use <input type="radio"> — select one option from a group.

7. Checkboxes

Use <input type="checkbox"> — select multiple options.

Interests:

8. Select Dropdown

Use <select> with <option> elements.

9. Textarea

Use <textarea> for multi-line text input.

10. Submit Button

Use <button type="submit"> to submit the form.
