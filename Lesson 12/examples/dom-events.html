🔍 Selecting Elements

Before you can change an element, you need to grab it. JavaScript provides multiple ways to select nodes in the DOM tree.

💡 Pro tip: querySelector and querySelectorAll are the modern standard. They take any CSS selector (like .class or #id)!

✏️ Manipulating Content & Styles

Once you select an element, you can change its text (textContent), HTML contents (innerHTML), or its CSS classes (classList).

Property / MethodDescription
element.textContent = 'Hi'Changes exact text inside the element. (Safe from XSS)
element.innerHTML = '<b>Hi</b>'Changes inner HTML structure. (Use cautiously)
element.classList.add('active')Adds a CSS class to the element.
element.classList.toggle('dark')Toggles a CSS class off and on.

🖱️ Event Listeners

Events happen when the user interacts with the page (clicks, typing, hovering). We use addEventListener to execute functions when these events occur.

button.addEventListener('click', function(event) {
    console.log('Button clicked!');
});
