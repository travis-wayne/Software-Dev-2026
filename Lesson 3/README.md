# Lesson 3 – First Project: Personal Portfolio (Structure)

## 📚 Learning Objectives

By the end of this session, you will be able to:

1. Plan the structure of a multi-page personal portfolio website.
2. Create the basic HTML file structure for a personal portfolio.
3. Link multiple HTML pages together using navigation.
4. Apply semantic HTML5 elements to organize portfolio content.

## 🗂️ Folder Structure

```
Lesson 3/
├── README.md                       ← You are here
├── notes/
│   ├── tutor_notes.md              ← Tutor guide, session outline & tips
│   └── student_notes.md            ← Student study notes & resources
├── examples/
│   └── wireframe_outline.md        ← Example wireframe & content outline
├── exercises/
│   └── personal-portfolio/         ← Starter project folder
│       ├── index.html              ← Home page (started)
│       ├── about.html              ← About Me page (scaffold)
│       ├── projects.html           ← Projects page (scaffold)
│       ├── contact.html            ← Contact page (scaffold)
│       └── images/                 ← Folder for portfolio images
└── assignments/
    └── assignment_brief.md         ← Post-session assignment details
```

## 🔑 Key Concepts

- **Project Planning** — Wireframing, content outlines, breaking down a project
- **Multi-page Websites** — Linking separate HTML files together
- **Navigation** — `<nav>`, `<a>` tags, consistent menus across pages
- **Semantic HTML5** — `<header>`, `<main>`, `<footer>`, `<section>`, `<article>`, `<aside>`
- **File Organization** — Separate folders for HTML, images, styles
