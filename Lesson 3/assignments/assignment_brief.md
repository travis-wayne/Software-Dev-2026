# Assignment – Lesson 3: Complete Your Personal Portfolio Structure

## Objective

Complete the basic HTML structure for all pages of your personal portfolio website.

## Requirements

### All Pages Must Have:

- [x] Proper HTML5 boilerplate (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- [ ] A `<header>` with site title and navigation
- [ ] A `<nav>` bar with links to **all four pages**
- [ ] A `<main>` section with page-specific content
- [ ] A `<footer>` with copyright info
- [ ] Placeholder content (real content can come later)

### Page-Specific Tasks:

#### `index.html` (Home)
- [ ] Welcome message with your name
- [ ] Brief introduction about yourself
- [ ] (Optional) A "Skills" or "Featured" section

#### `about.html` (About Me)
- [ ] A paragraph about yourself
- [ ] Education or background section
- [ ] Hobbies or interests section

#### `projects.html` (Projects)
- [ ] At least **3 project entries** using `<article>` tags
- [ ] Each project should have a title (`<h3>`) and description (`<p>`)

#### `contact.html` (Contact)
- [ ] A working contact form with:
  - Name (text input)
  - Email (email input)
  - Subject (dropdown with at least 3 options)
  - Message (textarea)
  - Submit button
- [ ] Proper `<label>` tags on all fields
- [ ] `required` attribute on important fields

### Testing Checklist:
- [ ] Open `index.html` in the browser
- [ ] Click every navigation link — do they all work?
- [ ] Does every page have the same navigation bar?
- [ ] Does the browser tab show the correct page title?

## Bonus Challenges ⭐

- Add a profile image to the About page using `<figure>` and `<figcaption>`.
- Add links to external project demos or GitHub repos on the Projects page.
- Add social media links to the footer.
- Try to make the current page link look different in the nav (we'll cover CSS next!).

## Submission

Make sure all files are saved in your `personal-portfolio/` folder and test everything in the browser before the next session.
