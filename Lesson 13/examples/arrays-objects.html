📚 What is an Array?

An array is a single variable that holds a list of values. Instead of creating dozens of separate variables, one array can hold an unlimited number of items — all accessible through a single name.

⚠️ Zero-Indexed: JavaScript always starts counting at 0, not 1. The first item is at index [0], the second at [1], and so on.

🔬 Visual: How Array Indexes Work

Every item has an exact numbered position — its index. Think of it like seat numbers on a bus that start from seat 0.

🗂️ What is an Object?

An object stores data as key-value pairs. Instead of using numbers to find data (like arrays), you use descriptive names (keys). This makes data self-describing and far easier to read.

💡 Analogy: An array is like a shoebox where items are numbered by position. An object is like a labelled filing system — each drawer has a name like "Name", "Age", "Email".

🔬 Visual: Key-Value Structure

Each property is a pair — a key (the label) maps to a value (the data):

🔄 Higher-Order Array Methods

These methods accept a function as an argument (called a callback) and apply it to every item in the array. They are the modern, readable replacement for manual for loops when processing data.

⚠️ Key Rule: Both .map() and .filter() ALWAYS return a brand new array. They never change the original.
MethodWhat it doesReturns
.map(fn)Transforms every itemNew array of same length
.filter(fn)Keeps only items where fn returns trueNew array, possibly shorter
.forEach(fn)Runs fn for each item (side-effects only)undefined — use for logging, not data

✨ Why ES6+ Matters

ES6 (released 2015) modernised JavaScript. These three features alone will make your code dramatically more readable, concise, and professional. Every JavaScript developer uses them daily.

A. Template Literals — No More String Pasting

Use backticks (the key above Tab) and ${} to embed variables directly in strings.

❌ Old Way (confusing)
const n = "Sarah";
const s = 95;
console.log(
  "Hello " + n
  + ", score: "
  + s + "/100"
);
✅ Template Literal
const n = "Sarah";
const s = 95;
console.log(
  `Hello ${n}, score: ${s}/100`
);

B. Arrow Functions — Concise Function Syntax

Arrow functions replace the function keyword. When the body is a single expression, curly braces and return can both be dropped.

❌ Old (verbose)
function double(n) {
  return n * 2;
}
✅ Arrow (concise)
const double = n => n * 2;

// With two params:
const add = (a, b) => a + b;

C. Destructuring — Unpack Values Instantly

Destructuring lets you extract multiple values from an object or array in a single, readable line.

❌ Old (repetitive)
const title = product.title;
const price = product.price;
const brand = product.brand;
✅ Destructuring (one line)
const { title, price, brand }
  = product;
