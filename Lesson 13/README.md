# Lesson 13: JavaScript Fundamentals II – Arrays, Objects, ES6+ Features

## 🎯 Learning Objectives

By the end of this session, students will be able to:
1.  **Understand and manipulate Arrays:** Create ordered lists of data, access elements by index, and use array methods (`push`, `pop`, `map`, `filter`).
2.  **Model data with Objects:** Create collections of key-value pairs to represent real-world entities and access properties using dot and bracket notation.
3.  **Apply modern ES6+ syntax:** Write cleaner code using Arrow Functions, Destructuring Assignment, and Template Literals.

## 🧠 Key Concepts

*   **Arrays:** Ordered collections accessed via zero-based indices. Ideal for lists (e.g., shopping list, high scores).
*   **Objects:** Key-value stores accessed via string keys. Ideal for structured data (e.g., a specific User, Product).
*   **Arrow Functions:** `const add = (a, b) => a + b` (Concise syntax, implicitly returns).
*   **Destructuring:** `const { name, age } = user;` (Unpacks properties or array items directly into variables).
*   **Template Literals:** `` `My name is ${name}` `` (String interpolation with backticks).
