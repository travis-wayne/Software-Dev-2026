# Assignment – Lesson 4: Style Your Personal Portfolio with CSS

## Objective

Create an external stylesheet for your personal portfolio and apply basic CSS styling to all pages.

## Requirements

### Setup:

- [ ] Create a new file called `portfolio.css` in your `personal-portfolio/` folder.
- [ ] Link `portfolio.css` to **all** HTML pages (`index.html`, `about.html`, `projects.html`, `contact.html`) using `<link>`.

### Body & Typography:

- [ ] Set a default font for the `body` (e.g., `font-family: 'Segoe UI', Tahoma, sans-serif;`)
- [ ] Set a default text color for the `body` (e.g., `color: #333;`)
- [ ] Set a background color for the page.

### Headings:

- [ ] Style `h1` headings with a specific color and size.
- [ ] Style `h2` headings with a different color and/or size.
- [ ] (Optional) Style `h3` headings if used on your Projects page.

### Spacing — Box Model:

- [ ] Add `padding` to your `<main>` or `<section>` elements for breathing room.
- [ ] Add `margin` between sections or articles so content doesn't feel cramped.
- [ ] (Optional) Add a `border` to at least one element (e.g., project cards, contact form).

### Navigation:

- [ ] Style your `<nav>` links with custom colors.
- [ ] Add a `:hover` effect so links change color when the mouse hovers over them.
- [ ] Remove the default underline on links using `text-decoration: none;`.
- [ ] (Optional) Add a background color or padding to make the nav look like a bar.

### Contact Form (if applicable):

- [ ] Style `<input>` and `<textarea>` fields with padding and borders.
- [ ] Style the submit button with a background color and hover effect.

### Testing Checklist:

- [ ] Open each page in the browser — do the styles apply consistently?
- [ ] Hover over links — does the `:hover` effect work?
- [ ] Right-click → Inspect — can you see the Box Model for your styled elements?
- [ ] Are your styles organized and easy to read in `portfolio.css`?

## Bonus Challenges ⭐

- Add a `:focus` style to your form inputs (e.g., a colored border when clicked).
- Use a Google Font (e.g., `Inter`, `Roboto`) by adding a `<link>` in the `<head>`.
- Add a subtle background color to your `<header>` and `<footer>`.
- Try the `box-sizing: border-box;` rule and see how it changes element sizing.
- Style your project entries as "cards" using padding, border, and background.

## Submission

Make sure `portfolio.css` is saved in your `personal-portfolio/` folder, linked to all pages, and all styles are visible in the browser before the next session.
