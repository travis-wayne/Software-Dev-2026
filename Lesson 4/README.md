# Lesson 4 – CSS Basics, Box Model, Selectors

## 📚 Learning Objectives

By the end of this session, you will be able to:

1. Understand the purpose of CSS and how it styles HTML elements.
2. Apply CSS using inline, internal, and external stylesheets.
3. Explain and apply the CSS Box Model (content, padding, border, margin).
4. Use various CSS selectors (element, class, ID, attribute, pseudo-class) to target specific HTML elements.

## 🔑 Key Concepts

- **CSS (Cascading Style Sheets)** — The language used to style and layout web pages
- **Including CSS** — Inline, Internal (`<style>`), and External (`<link>`) methods
- **CSS Syntax** — Selectors, properties, and values
- **The Box Model** — Content, Padding, Border, Margin
- **CSS Selectors** — Type, Class, ID, Universal, Attribute, Pseudo-classes
- **Specificity** — How the browser decides which styles to apply
