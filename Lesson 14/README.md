# Lesson 14: Asynchronous JavaScript (Promises, Async/Await & Fetch)

## Overview
This lesson introduces the concept of **Asynchronous programming** in JavaScript. We explore how to handle tasks that take time (like network requests) without freezing the user interface, moving from the old callback patterns to modern **Promises**, **Async/Await**, and the **Fetch API**.

## Learning Objectives
By the end of this session, students will be able to:
- Explain the difference between **Synchronous** (blocking) and **Asynchronous** (non-blocking) code.
- Create and consume **Promises** using `.then()`, `.catch()`, and `.finally()`.
- Use **Async/Await** syntax to write cleaner asynchronous code.
- Use the **Fetch API** to retrieve and display data from external public APIs.
- Implement robust **Error Handling** for network requests.
