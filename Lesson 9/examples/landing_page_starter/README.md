# DevSync Landing Page

This is the central repository for the collaborative "DevSync" Landing Page project.

## How to Contribute

1. Check the Kanban/Project board to **claim a ticket** (e.g., Ticket #2: Hero Section).
2. Fetch the latest codebase: `git checkout main` -> `git pull origin main`.
3. Create your feature branch exactly named for the feature: `git checkout -b feature/build-hero`.
4. Open `index.html`. **ONLY write code inside the HTML comments designated for your section.** Do not modify the rest of the file.
5. Use Tailwind CSS utility classes for styling.
6. Commit often and push your branch: `git push -u origin feature/build-hero`.
7. Open a Pull Request on GitHub and request a review from your Lead Developer/Tutor.

## Current Setup
- Semantic HTML shell
- Tailwind CSS loaded via CDN
- Blank custom CSS file ready in `css/`

## Golden Rule
Never push directly to `main`!
